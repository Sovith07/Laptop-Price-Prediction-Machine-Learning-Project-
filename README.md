# Laptop-Price-Prediction-Machine-Learning-Project-
This project predicts laptop prices based on hardware specifications using machine learning models and feature engineering.

Features Used

Company & Type
RAM & Weight
Touchscreen & IPS Display
PPI (Pixel Density)
CPU Brand
GPU Brand
Storage (SSD/HDD)
Operating System


ML Models Used

Linear Regression, Ridge, Lasso
KNN, Decision Tree
Random Forest, Gradient Boosting
SVR, XGBoost
Ensemble Models (Voting, Stacking)

Best Model Performance
Model: XGBoost
R² Score: 0.896
MAE: 0.154 (log-scale)


Key Insights
SSD significantly increases laptop price
High PPI displays are premium
Nvidia GPUs and Intel i7 CPUs strongly impact pricing
Touchscreen & IPS panels increase cost


Tech Stack
Python
Pandas, NumPy
Scikit-learn
XGBoost
Seaborn, Matplotlib
